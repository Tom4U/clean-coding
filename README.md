# Beispiele und Übungen zum Clean Code Seminar

Die meisten Beispiele und Übungen sind aktuell in Java und werden bei Bedarf erweitert um zusätzliche Sprachen.

Die Ordner `demos`, `challenge` und `solution` sind in Visual Studio Code erstellte Maven Projekte inklusive Launch-Konfigurationen.

Die Lösungen im Projekt `solution` haben standardmäßig `checkstyle` aktiv und die Regeln werden in Visual Studio Code automatisch geprüft.
