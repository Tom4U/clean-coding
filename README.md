# Beispiele und Übungen zum Clean Code Seminar

Die meisten Beispiele und Übungen sind aktuell in Java und werden bei Bedarf erweitert um zusätzliche Sprachen.

Die Ordner `demos`, `uebungen/challenges`, `uebungen/loesungen`, `finale/challenge` und `finale/loesung` sind in Visual Studio Code erstellte Maven Projekte inklusive Launch-Konfigurationen (wo anwendbar).

Die Lösungen im Projekt `uebungen/loesungen` haben standardmäßig `checkstyle` aktiv und die Regeln werden in Visual Studio Code automatisch geprüft.
